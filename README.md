# Climate Data Web Application
This was my assignment for Year 1 Semester 1 Java Studio class.

## Description
This project was developed as part of a group assignment. 

## Project Context
This project is a web based application showcasing climate data analytics. 
It was done in a group of three, where each one of us had three webpages to design and code. 
I have removed their pages from this code, and removed their names.

## My Contribution
I created the homepage, which displays a summary of the climate data, and has icons which work as links to other pages.
I also created the weather station analytics page, which allows for the user to filter weather stations with certain metrics.
I also created the weather station similiarities page, which allows for the user to find weather stations with similiar metrics.

## Technologies
- Java
- SQL
- HTML
- CSS

## How to Run
	1.	Clone the repository
	2.	Open the project in VS Code or IntelliJ
	3.	Run the main application class (App.java)
	4.	Open browser at: http://localhost:7001/
